MySQL

¿Que es MySQL?

MySQL es un software de open source desarrollado por Oracle. Permite almacenar y acceder a datos a traves de diferentes motores de almacenamiento, tambien es capaz de replicar datos y distribuir tablas para mejorar el rendimiento y la durabilidad. Gestiona bases de datos utilizando el lenguaje de consulta SQL

Ventajas de MySQL:

  1. Open source
  2. Compatibilidad: Es compatible con Linux, MacOS, Microsoft windows y Ubuntu
  3. Facilidad de uso: Su interfaz grafica permite asocir utilidades del programa rapidamente.
  4. Soporte comunitario

¿Que es MySQL Workbench?

MySQL Workbench es una herramienta gráfica para trabajar con servidores y bases de datos MySQL.

Principales Funciones de MySQL Workbench:

  1. Desarrollo de SQL. Permite crear y administrar conexiones a servidores de bases de datos. Además de permitirle configurar los parámetros de conexión, MySQL Workbench brinda la capacidad de ejecutar consultas SQL en las conexiones de la base de datos utilizando el Editor SQL incorporado.
  2. Diseño de Datos. Permite crear modelos del esquema de su base de datos gráficamente, ingeniería inversa y directa entre un esquema y una base de datos en vivo, y editar todos los aspectos de su base de datos utilizando el editor de tablas completo. El editor de tablas proporciona funciones fáciles de usar para editar tablas, columnas, índices, disparadores, particiones, opciones, inserciones y privilegios, rutinas y vistas
  3. Administracion del servidor. Permite administrar las instancias del servidor MySQL mediante la administración de usuarios, la realización de copias de seguridad y recuperación, la inspección de datos de auditoría, la visualización del estado de la base de datos y la supervisión del rendimiento del servidor MySQL.
  4. Migracion de Datos. Permite migrar desde Microsoft SQL Server, Microsoft Access, Sybase ASE, SQLite, SQL Anywhere, PostreSQL y otras tablas, objetos y datos de RDBMS a MySQL. La migración también admite el cambio de versiones anteriores de MySQL a las versiones más recientes.
  5. Compatibilidad con MySQL Enterprise. Compatibilidad con productos empresariales como MySQL Enterprise Backup, MySQL Firewall y MySQL Audit.
